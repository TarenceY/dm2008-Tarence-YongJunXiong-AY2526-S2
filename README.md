## Projects

### Mini Project — Flappy Bird
A Flappy Bird clone in p5.js. Features a start screen, bird rotation, scrolling background, sound effects, and a persistent high score via `localStorage`.
→ [02-mini-project/README.md](02-mini-project/README.md)

### Final Project — Unimpressed Blob
An interactive pixel-art character (Talking Ben / Minecraft-style) that reacts to physical force input via an Arduino force sensor. It cycles through 10 emotional states, supports two skins, a secret combo dance, mouse-throw physics, and full sound design.
→ [04-final-project/README.md](04-final-project/README.md)
